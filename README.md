# Sistema-de-Auditor-a-y-Reportes-Regulatorios-Fintech
Este proyecto procesa y almacena transacciones de pago para el cumplimiento de normativas financieras. Garantiza la total inmutabilidad de los registros, transacciones con propiedades  y permite auditorías históricas exactas. El sistema genera de manera automatizada los reportes regulatorios fijos a fin de cada mes.
